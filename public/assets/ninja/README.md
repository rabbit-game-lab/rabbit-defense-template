# Ninja Adventure visual subset

This directory contains a deliberately small visual subset derived from the **Ninja Adventure Asset Pack** by Pixel-Boy and AAA.

- Source: https://pixel-boy.itch.io/ninja-adventure-asset-pack
- Retrieved: 2026-07-23
- Source archive SHA-256: `95a06f4fdcfd1882f061a45ff313b7c905dbe2de1e8512b281d7937df62a7b15`
- License: CC0 1.0 Universal; see [`LICENSE.txt`](./LICENSE.txt)
- Runtime size: approximately 64 KB instead of redistributing the complete pack

## Included derivatives

- `world-board.png`: composed from `TilesetFloor.png`, `TilesetField.png`, and `TilesetNature.png` while preserving the template's gameplay path.
- `tower-*.png` and `hidden-dojo.png`: 32×32 crops from `TilesetTowers.png`.
- `enemy-*.png`: 16×16 idle frames from the Mouse, Racoon, Panda, and Bear monster sheets.
- `projectile-*.png`: first frames from Shuriken, IceSpike, and CanonBall projectile sheets.

The source gameplay identifiers (`arrow`, `frost`, `bomb`, etc.) intentionally remain stable for save/rules compatibility; only player-facing theme names and presentation change.
